Ooze

An Aqua-inspired desktop for Linux. Built on Mutter, with first-party apps that share one look.

Download AppImage Download .deb

Files, terminal, and one-click dark mode.

What you get

A complete shell

Global menu bar, floating dock, and desktop icons on a Wayland compositor.

One design language

File manager, terminal, viewer, and settings — all first-party, all matching.

Light and dark

Switch the whole desktop from the Ooze menu. Every app follows instantly.

Runs your apps

Any Wayland or X11 application works, via built-in Xwayland support.

Screen sharing

First-party Ooze ScreenCast portal: share a monitor from Flatpak Discord or browser getDisplayMedia. Window share and screenshots to come.

Default applications

Set browser, mail, and other defaults in Ooze King.

Self-contained

WhiteSur themes and elementary icons are bundled.

Two tracks

Log into the stable .deb; iterate with ./run-devkit.sh in a nested window. See DEVELOPING.md.

Light and dark — same desktop, one click apart.

Try it

Grab the AppImage or .deb from Releases — AppImage runs nested on any Wayland desktop:

chmod +x Ooze-v0.7.0-x86_64.AppImage && ./Ooze-v0.7.0-x86_64.AppImage

Or install the native session package:

sudo apt install ./ooze_0.7.0_amd64.deb

Download ooze_0.7.0_amd64.deb

Or build from source (Mutter 18 dev packages, e.g. Ubuntu 26.04):

git clone https://github.com/Zadagast/ooze
cd ooze && ./run-devkit.sh
